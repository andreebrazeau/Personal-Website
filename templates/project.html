Portfolio

MuniMobile is the second place winner project of the Linkedin DevelopHer Hackathon presented at the end of June2012. MuniMobile is a web application that will text you schedules, real-time updates of Muni buses approching your stop.

Built on Django. We used NextBus and Twilio APIs. All the backend has beeen developed in Python. The front end in JavaScript and JSON.

I wrote a good part of the backend. http://munimobile.herokuapp.com
